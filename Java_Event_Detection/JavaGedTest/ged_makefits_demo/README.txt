This dirs contains 3 mains packages for ged makefits demo.

1) Control Center 
2) Makefits for aircraft tracks
3) Makefits for ship tracks 

The demo shows how to apply ged to track applications. There are two track 
domains. One for ships and aircrafts. The control center can subscribe to 
the events of interest from the remote site.


It also includes the following bat files


1) startDemo.bat :: for starting the demo

FOR DEVELOPERS USE:
2) removebak.bat :: for removing back up file
